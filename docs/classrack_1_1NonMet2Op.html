Detailed Description

Marks bins with low RhoHV value as probable anomalies.

Constructor & Destructor Documentation

◆ NonMet2Op()

NonMet2Op ( const drain::UniTuple< double, 2 > &  threshold = {0.75,0.95},
const drain::UniTuple< double, 2 > &  medianWindow = {0.0,0.0},
double  medianThreshold = 0.95 
)

Default constructor.

Parameters
threshold- RhoHV values below this will be considered this anomalies
thresholdWidth- steepness coefficient for the threshold
windowWidth- optional: median filtering window width in metres
windowHeight- optional: median filtering height width in degrees
medianPos- optional: median position: 0.5 = normal median ( >0.5: conservative for precip) //(morphology: 0.5 > opening; 0.5 < closing)

This operator is universal , it is computed on DBZ but it applies also to str radar parameters measured (VRAD etc)
