Splits and trims a given std::string to a std Sequence.

Splits and trims a given std::string to two substrings, by first separator encountered.

Template Parameters
T1- target of 1st part ("key")
T2- target of 1st part ("value")
C- character or string (any char in which will split)
Parameters
s- source string
first- target of 1st part, typically the "key"
second- target of 2nd part, typically the "object"

Given an empty std::string, returns empty (intact) destinations. Given a string that does contain a separator, assigns the string on first , leaving empty/intact.

Note: the classes of target objects first and second should have an assignment operator=() that does not clear the object when assigning itself obj=obj !
